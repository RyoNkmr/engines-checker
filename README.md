# Engines Checker

## Requirements

- Node.js >= 8.0

## Example

### package.json
```json
{
  "name": "some-projects",
  "version": "0.0.1",
  "engines": {
    "node": ">=9.9.0",
    "npm": ">=5.6.0"
  },
  "scripts": {
    "start": "..."
  }
}
```

## Why dont you use Docker? / Motivation

Most of the frontend developers works with me strongly loves thier local environments though i created docker images for some projects.
